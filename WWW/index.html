GoStyle Project

The aim of the project is to investigate the possibilities of computer analysis of databases of Go game (Baduk, Weichi) records using computer data-mining algorithms.

What is here for players?

GoStyle Web App is a web application which analyzes your Go game records, determines your playing style, strength, and gives various study hints, including a list of relevant professional players to review and book recommendations.
